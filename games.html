Where Wulf
a killer party game

A recreation of my favorite card game, One Night Ultimate Werewolf. Full credit to Bezier Games. I wanted to make a slightly faster version of the game didn't require everyone closing their eyes. My version instead displays hidden information to an individual's phone. I took some artistic liberties but I beleive the original rules are still intact and anyone who has played the original could play my version with minimal issues.

Where Wulf was created with a React front-end that talks to my REST API that was developed in Java using Spring. Both parts are hosted on Heroku. I still have a lot of ideas for this project. I would like to setup a database to store the outcome of games so that I could see stats. I also would like to add some of the expantion packs.
